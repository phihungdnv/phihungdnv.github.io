description: chown command option

```
Change the owner of file
	chown phihung myfile.txt

Change the owner and group
	chown user:group myfile.txt

Change the owner and group file and directory
	chown -R user:group /folder
	
Change user but not change group
	chown user: file
	
Change group but not change user
	chown :group file
	
```
